• Explain why do we want sometimes to use setImmediate instead of using setTimeout?

    When we have IO on event loop's 4th queue and we want to run function before close event is called we can use setImmediate. If event loop's queues are empty then setTimeout and setImmediate will called in order of event loop.

  • Explain the difference between process.nextTick and setImmediate?

    Difference is nextTick's function will be called immediately after finishing current function regardless of event loops queue because it has more priority. setImmediate will be called on 5th queue of event loop so it has less priority.

  • Name 10 core modules that Node provides by default.
    1. http - includes classes, methods and events to create Node.js http server.
    2. url - includes methods for URL resolution and parsing.
    3. querystring - includes methods to deal with query string.
    4. path - includes methods to deal with file paths.
    5. fs - includes classes, methods, and events to work with file I/O.
    6. util - includes utility functions useful for programmers.
    7. net - To create servers and clients.
    8. stream - To handle streaming data.
    9. tls - To implement TLS and SSL protocols.
    10. vm - To compile JavaScript code in a virtual machine.
